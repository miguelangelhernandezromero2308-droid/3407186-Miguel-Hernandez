# 📅 Proyecto Semana 1 – Ficha de Presentación de Dominio

## 📌 Información del Proyecto

Este proyecto corresponde a la **Semana 1 del Bootcamp de JavaScript (ES2023)**.  
El objetivo es crear un script en **JavaScript ejecutado con Node.js** que muestre en la consola una ficha de presentación de una entidad perteneciente a un dominio específico.

En este proyecto el dominio asignado es:

**Sistema de citas médicas online**

El programa muestra información básica de una **cita médica**, utilizando únicamente los conceptos aprendidos durante la primera semana.

---

# 🎯 Objetivo del Proyecto

Construir un script que:

- Muestre información en consola usando `console.log()`
- Utilice **tipos de datos primitivos**
- Incluya **comentarios en el código**
- Organice la información en **secciones claras**

Este proyecto funciona como una introducción a la programación con **JavaScript y Node.js**.

---

# 🧠 Conceptos Utilizados

En este proyecto se aplican los siguientes conceptos básicos de JavaScript:

- `console.log()` para mostrar información en consola
- **String** para representar texto
- **Number** para valores numéricos
- **Boolean** para representar estados verdadero/falso
- **Expresiones aritméticas**
- **Comentarios en el código**
- Organización de la salida en consola

---

# 🏥 Dominio del Proyecto

**Dominio:** Sistema de citas médicas online  

**Entidad principal:** Cita médica

El sistema representa una cita médica con información como:

- Nombre de la consulta
- Código de la cita
- Categoría de la consulta
- Precio de la consulta
- Estado de la cita
- Disponibilidad

# ▶️ Cómo Ejecutar el Proyecto

1. Abrir la terminal.
2. Ubicarse en la carpeta del proyecto.
3. Ejecutar el siguiente comando:

```bash
node script.js
```

El programa mostrará en la consola una **ficha informativa de una cita médica**.
