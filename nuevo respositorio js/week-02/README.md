## 📖 Descripción del Proyecto

En esta segunda semana se continúa con el desarrollo del proyecto basado en el dominio **Sistema de Citas Médicas Online**.  
El objetivo es seguir fortaleciendo los conocimientos en **JavaScript**, aplicando estructuras del lenguaje para representar procesos básicos dentro de un sistema de gestión de citas médicas.

Este ejercicio simula acciones que podrían realizarse dentro de una plataforma de salud digital, como el manejo de información de pacientes, doctores o citas médicas.

---

## 🎯 Objetivo

Aplicar conceptos fundamentales de **JavaScript** para desarrollar lógica básica que represente funcionalidades de un sistema de citas médicas online.

## ▶️ Ejecución del Proyecto

Para ejecutar el proyecto sigue estos pasos:

1. Tener instalado **Node.js** o un navegador web.
2. Descargar o clonar el repositorio.
3. Abrir el archivo `script.js`.
4. Ejecutar el archivo desde la terminal con:

```
node script.js
```

o ejecutarlo desde la consola del navegador.
